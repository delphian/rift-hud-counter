AOM Rift Counter

Small window that displays the current amounts of all currencies and experience in Rift. Second column displays changes since the last counter reset. Any achievements you progress in while playing will also be displayed in the counter.

Installation

First locate Rift's addon directory:

  1. From rift activate the main menu by pressing the 'esc' key.
  2. Click on the Addons button.
  3. Click on the Open Addons Directory button.

Download the zip file and extract into Rift's addon folder

  • Download AOMCounter-2.1-0.5.1.zip
  • When the download is finished click the options arrow for the downloaded file at the bottom of your browser and select show in folder
  • Drag and drop the zip file into the Rift addon folder.
  • If you have a previous version of AOMCounter in the addons folder then remove the existing AOMCounter folder first.
  • Right click on the zip file now located in the addons folder and select extract here.

Enable the addon:

Now restart Rift, refresh the addons and enable the AOMCounter. You may need to restart Rift one more time.

Usage

  • Start the counter: /aom init
  • Reset the counters: /aom reset
  • Show the counter /aom show
